ReadLite · 使用手册

三分钟上手沉浸式阅读

界面风格与阅读器一致，确保你看到的效果与真实阅读体验相同。

1. 进入阅读模式

  1. 在浏览器扩展管理中将 ReadLite 固定到工具栏。
  2. 打开一篇网页文章，直接点击工具栏的 ReadLite 图标即可进入阅读模式。
  3. 进入后，页面会出现浮动控制条，可随时收起或关闭阅读视图。

2. 控制条速览

  • 主题：6 套内置主题，点击即刻预览。
  • 字体：任意选择（含等宽），即时作用于正文与代码。
  • 字号 / 行距：滑动或点选，实时生效。
  • 宽度：30 / 50 / 70 / 100 四档，适配不同屏幕与姿势。
  • 对齐：左对齐 / 两端对齐自由切换。
  • 关闭：右上角 “×” 可退出阅读模式。

3. 主题与排版

  • 默认主题：纸墨单色（低对比、接近墨水印刷）。
  • 其他可选：夜读、竹露、天青、冷紫、纯黑——点击即刻预览。
  • 排版：宽度 30/50/70/100%，左对齐 / 两端对齐可切换；行距、字号实时生效。

4. 字体

  • 字体由你决定，列表内含中英文通吃与多款等宽字体（JetBrains Mono、FiraCode Nerd Font 等）。
  • 内置字体：LXGW WenKai、Noto Serif / Noto Sans、Source Serif 4、Literata、Noto Sans Mono、JetBrains Mono、IBM Plex Mono、FiraCode Nerd Font Mono（同时可用系统字体栈）。

5. 高亮与导出

  • 选中文本即可高亮或写笔记，支持导出 Markdown（如需下载权限会提示）。
  • 高亮颜色随主题调整，保证夜间与深色模式的可读性。

6. 隐私与权限

  • 不采集、不上传任何阅读内容。
  • 权限仅用于在当前页注入阅读视图并保存本地设置；导出时如需下载会再次确认。
  • 设置保存在本地 storage，可随时删除扩展或清理浏览器数据以重置。

常见排障

  • 按钮没反应：刷新页面后重试；确保不是浏览器的内置特殊页。
  • Edge/Firefox 提示 “Failed to send toggle message”：切回目标标签页再点击，或重新打开标签。
  • 字体/主题没变化：在设置里重新选择并保存，确认页面未被站点脚本阻拦。
  • 内核限制：Chrome/Edge/Firefox 的扩展页面、PDF 预览等特殊页无法被接管。
